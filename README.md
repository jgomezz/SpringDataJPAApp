## Application SpringDataJPAApp


### Instalación de Maven - Windows 

1. Bajar la última versión de Maven desde https://maven.apache.org/download.cgi
2. Descomprimirlo en el directorio de ejecutables
3. Crear la variable de entorno del sistema MAVEN_HOME conteniendo el valor de la ruta de instalación de la aplicación.
4. Agregar en la variable de entorno del sistema PATH la siguiente ruta %MAVEN_HOME%\bin  

### Instalación de Terminal - Git Bash (Incluido dentro del instalador de Git)

1. Bajar la última versión de Git desde https://git-scm.com/
2. Instalar el ejecutable
